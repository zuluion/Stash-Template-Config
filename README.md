# Stash-Template-Config
> 自用Stash规则配置模板

[Stash-Template-Config](https://cdn.jsdelivr.net/gh/zuluion/Stash-Template-Config@master/Stash-Template-Config.yaml)

> 推荐使用Stash内置的Sub-Store本地管理机场订阅,只需修改以上配置文件中对应的Sub-Store订阅名称即可，也可以直接将以上配置文件的对应订阅地址改成机场订阅URL
